# loxfmt

loxfmt is a code formatter for the Lox programming language.

## Installation

```sh
go install github.com/marcuscaisey/lox/loxfmt@latest
```

## Usage

```
Usage: loxfmt [flags] [path]

Options:
  -help
        Print this message
  -print-ast
        Print the AST only
  -write
        Write result to (source) file instead of stdout
```
